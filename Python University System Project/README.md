# University Management System

A complete University Management System implemented in Python using Object-Oriented Programming principles. This system is designed for educational purposes and demonstrates proper software architecture, data management, and file handling.

## Features

### Core Entities
- **Students**: Manage student records, enrollments, and GPA calculations
- **Faculty**: Manage faculty members and their course assignments
- **Courses**: Manage course offerings and student enrollments
- **Departments**: Manage academic departments and their offerings
- **University**: Central management system with comprehensive statistics

### Functionalities
- ✅ Add, update, delete, and display all entities
- ✅ Student enrollment in courses
- ✅ Faculty assignment to courses
- ✅ GPA calculation and tracking
- ✅ Comprehensive search operations
- ✅ University statistics and reporting
- ✅ Data persistence with JSON file storage
- ✅ Input validation and error handling
- ✅ Sample data generation

### Advanced Features
- ✅ Sorting students by GPA
- ✅ Search by name or ID
- ✅ Department-wise statistics
- ✅ Grade management system
- ✅ Interactive console-based menu system
- ✅ Data backup and recovery
